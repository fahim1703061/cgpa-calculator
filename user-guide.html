User Guide

Welcome to the CGPA Calculator User Guide!

How to Use

  1. Enter your current CGPA.
  2. Enter the number of semesters you have completed.
  3. Enter your target CGPA.
  4. Enter the number of semesters after which you want to achieve the target CGPA.
  5. Click the "Calculate" button.

The calculator will then display the required GPA you need to earn in the next semester to achieve your target CGPA.
